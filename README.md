## Go高并发秒杀系统API
## How to play

 1. go get -u github.com/kongdingchao/goseckill
 2. 直接运行：goseckill
 3. 默认访问地址：localhost:8080

## Develop environment
  1. Beego:1.12.0
  2. go:1.13.5

## Written in front of words


完成这个秒杀系统，需要完成四个模块的代码编写，分别是:  

- 1.Go高并发秒杀APi之业务分析与DAO层代码编写
- 2.Go高并发秒杀APi之Service层代码编写。
- 3.Go高并发秒杀APi之Web层代码编写。
- 4.Go高并发秒杀APi之高并发优化。
